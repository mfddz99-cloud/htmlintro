# Northstar Tech Studio Landing Page

## Project Brief
This project is a professional, responsive landing page for a fictional technology consulting company. It includes a hero section, services, an about section, a contact form, and accessible navigation.

## Technologies Used
- HTML5
- CSS3
- Git and GitHub

## Setup Instructions
1. Open the project folder in VS Code.
2. Open [index..html](index..html) in a browser or with Live Server.
3. Edit the content or images as needed.

## Business Rationale
The page is designed to feel polished and professional, helping a fictional company present its services clearly and build trust with potential clients.
